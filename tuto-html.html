Contrairement à ce que beaucoup de monde pense, le HTML n’est pas un langage de programmation, mais de description. On ne programme pas en HTML, on décrit le contenu d’une page aux navigateurs qui la visitent, ces navigateurs se chargent ensuite de l’afficher.
Le HTML (pour HyperText Markup Language) est basé sur un système de balises et d'attributs. Si l’on veut afficher par exemple un paragraphe, on va venir créer une balise de début de paragraphe (on dit qu’on ouvre la balise), on tape le contenu du paragraphe, puis on termine en plaçant une balise de fin de paragraphe (on dit qu’on ferme la balise).
Exemple : Il existe un très grand nombre de balises. Nous ne les verrons pas toutes, mais la « bible » du html les regroupe toutes avec des exemples d’usage ici => w3schools.
Le tutoriel suivant va vous apprendre comment recréer la page que vous avez sous les yeux. TutoCeption.

Les Balises

Tout d'abord, pour ce qui est de l'éditeur de texte vous avez le choix! Ici, nous utiliserons SublimeText.(why not)
Il est primordial de commencer chaque programme en HTML par le <!DOCTYPE html> car c'est elle qui permet l'affichage d'une page web en précisant le type de document et la version html utilisée (ici HTML5).
Ensuite, il faut ouvrir une balise html et c'est ici que se trouvera majoritairement votre code pour le site.


Vient l'entête avec la balise <head> et le corps <body>.
  • Head: cette partie ne s'affichera pas sur la page, en revanche, elle indique le titre du site et son encodage. Lorsqu'on voudra lier un fichier .css avec un .html c'est dans le head qu'on se placera.
  • Body: tout ce qui se trouvera dans cette balise sera afficher sur la page web!

Les attributs

Les attributs viennent compléter les informations d'une balise

La balise <div>

La balise <div> (division du document) permet d'organiser le contenu du code sans représenter quelque chose en particulier. Elle est généralement utilisée pour grouper des élements entre eux et leur appliquer un style avec les attributs class ou id.

En programmation, il est toujours utile d'écrire des commentaires pour les personnes qui liront après nous. Evidemment, le commentaire ne sera pas afficher sur la page web : <!-- Commentaire -->

Les liens

En HTML, il est important de savoir faire des liens sur une page, c'est-à-dire insérer une image, un lien d'un site internet ou bien marquer un endroit précis de la page pour s'y rendre plus facilement.
Pour cela on va utiliser les balises suivantes:

  • Faire un lien vers un site internet: <a href="lien du site"></a>
  • Faire un lien permettant d'envoyer un email : <a href="mailto:adresse"></a>
  • Insérer une image : <img src="source de l'image"></img>
  • Hypertexte vers un endroit de la page : <a href="#mots" accesskey="p" title="nomDuParagraphe"</a>
  • Insérer un fichier vidéo (première méthode) : <object type="application" data="flvplayer.swf" width="taille" height="taille">
  • Insérer un fichier vidéo (deuxième méthode) : <video id="id de la vidéo" width="largeur" height ="hauteur"><source src="source" type="video/mp4"></video>

Les listes

Il est possible de lister des mots, de façon ordonnée ou non.
Avec la balise <ul> qui crée une liste non-ordonnée, ou <ol> pour l'ordonner. Puis la balise <li> pour créer un élément de liste.

Les tableaux

Pour créer des tableaux, utilisez la balise <table> dans laquelle vous utiliserez la balise <tr> qui désigne une ligne du tableau. Et pour afficher un message vous devrez utiliser les balises <th> ou <td>

Les formulaires

Pour créer des formulaires ce sera la balise <form>.A l'intérieur de cette balise, on y place <input type="" name=""> afin de déterminer les élements du formulaire.
La balise <input> est suivie de 3 types d'attributs:

  • text: qui permet d'entrer une ligne de texte(nom,prénom..)
  • radio: qui permet à l'internaute de choisir un choix parmis d'autres.
  • submit: qui permet de soumettre ce que la personne aura rempli dans les champs précédents.
